Geometry Dash rating code

Published on February 15, 2020

Describing a level is a pain. To fix this, I have created a system to shorten difficulty, stars, feature status, and even 2 player status into a short string.

I. Is the level featured?

If the level is epic, write down E-. If it's featured, write down F-. If it's neither of those, write down U-.
SPECIAL CASE: If it is a main level, whether it's on GD, Meltdown, SubZero, or World, write down R-.

II. Version

The major release that this level was uploaded in, without the decimal point. Should be between 10 and 21, 22 when it finally releases, which it won't :(

III. What is the difficulty?

Depending on the difficulty, you should write down one of these 2-letter codes and then a hyphen:

  • NA - NA
  • Auto - AU/li>
  • Easy - EZ
  • Normal - NM
  • Hard - HA
  • Harder - HR
  • Insane - IS
  • Demon (Any Difficulty) - DM (This should always be used when referring to RobTop's demons)
  • Easy Demon - ED
  • Medium Demon - MD
  • Hard Demon - HD
  • Insane Demon - ID
  • Extreme Demon - XD

IV. Stars

This is simply the number of stars.

V. 2-Player Status

If the level has 2-Player mode enabled, add -2 to the end.

Examples

1st level (ID 128): U-10-NM-0
Bloodlust (ID 42584142): E-21-XD-10
Clubstep (Official Level): R-16-DM-14
Press Start (SubZero Level): R-22-NM-4
Horntail Cave (ID 7005127): F-19-ID-10-2
